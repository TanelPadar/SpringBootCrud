### Project setup

**To run react**

In the terminal navigate to src/frontend and type:

**npm install**

**npm start**

To run Spring Backend
In your IDE run PadarApplication.java

### App Functionalities

**GET** a list of users 

**CREATE** a new user

**EDIT** any user

**DELETE** a  user from the table
